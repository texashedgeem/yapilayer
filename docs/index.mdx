Yapilayer is an open-source Open Banking platform — a secure, extensible, standards-compliant foundation for connecting to financial institutions, starting with UK Open Banking.

<Note>
  This documentation site is a skeleton (Milestone 0). Full content — getting
  started, quickstart, AIS/PIS guides, provider development, deployment — is
  written in Milestone 10. See the repository's `ROADMAP.md` for the plan.
</Note>

## What Yapilayer provides

- **Account Information Services (AIS)** — consent management, account discovery, balances, transactions
- **Payment Initiation Services (PIS)** — payment creation, authorisation journeys, status tracking, webhooks
- **A provider plugin framework** — add a bank by implementing a connector module, never by changing the core
- **Generated SDKs** — Java and TypeScript, generated from the OpenAPI specification
